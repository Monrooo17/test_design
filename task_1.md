### Equivalence classes.

## 1. Валидные значения. 

Согласно ТЗ - поле принимает любые значения.

Пример ввода:

1. Латиница: adafADSF
2. Кириллица: аЫВЫФаыфв
3. Числа: 213414
4. Спец. символы: @#!$!@^&



## 2. Пустое значение. 

В поле логина ничего не вводить.

## 3. Строка с пробелами (начало/середина/конец)

Ввод:   ada dad sadasa  

## 4. Unicode символы.

Ввод: эмодзи, иероглифы
